# IIRABM

The innate immune response agent-based model (IIRABM)

## Getting started

### Installation

Install using cmake:

```
cmake CMakeLists.txt
make
```

### Example usage

In Python3:

```
import sepsis
sepsis.run({}) # Run a simulation of the IIRABM
```
